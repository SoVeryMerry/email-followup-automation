# Email Follow-up Automation

A desktop app that automates sending follow-up emails, integrated with Exchange for reading and sending mail.

## What it does

Runs as a desktop application (see followup_desktop_app.py) that automates the follow-up email process, reducing manual email work.

## Tech

Python, exchangelib for Exchange/Outlook integration, requests for API calls, BeautifulSoup for email content parsing.

## Setup

Install dependencies with pip install -r requirements.txt, then run followup_desktop_app.py.
